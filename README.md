# 🛒 E-Commerce Sales Analysis

Analisis data penjualan e-commerce menggunakan Python.

## 🛠️ Tools
- Python, Pandas, Seaborn, Matplotlib

## 🔍 Key Findings
- Technology menyumbang 64.8% total profit
- Region West margin tertinggi 14.4%
- Diskon ≥30% rata-rata rugi $136/transaksi

Source: [Superstore Sales Dataset](link Kaggle kamu di sini)
